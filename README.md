# Image Segmentation using K-Means Clustering

This project demonstrates image segmentation using K-Means clustering with Python and OpenCV.

## Features Implemented
- Image reading and preprocessing
- RGB color conversion using OpenCV
- Pixel reshaping for clustering
- K-Means clustering for image segmentation
- Visualization of segmented outputs
- Elbow method for determining optimal clusters

## Technologies Used
- Python
- OpenCV
- NumPy
- Matplotlib
- Scikit-learn

The objective of this project is to understand clustering techniques and apply K-Means for image segmentation tasks.
